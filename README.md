# Кейс: Система учета абитуриентов вуза "Анализатор поступления"



## Краткое описание проекта

Веб-приложение для автоматизации процесса учета абитуриентов высшего учебного заведения.
Система позволяет регистрировать абитуриентов, отслеживать поданные документы,
управлять специальностями и формировать отчеты для приемной комиссии.



## Инструкция по установке/развертыванию

1. Клонирование репозитория
2. Установка Node.js версии 22.22.0
3. Войти в деректорию
4. Написать в консоли node server.js для запуска сервера
